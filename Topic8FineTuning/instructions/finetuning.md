# Topic 8: Fine-Tuning an LLM

![llama tuning TV](finetuning.jpg)

### Learning Goals

- Understand different varieties of fine-tuning
- Know when fine-tuning is more appropriate than RAG
- Know how to run a fine-tuning pipeline using Tinker.

### Tasks

1. Find your [partner on this sheet](https://docs.google.com/spreadsheets/d/1tHrIawCK6SmvuhWP3AJWWpYONON0LQ3ne8IBw3iotBM/edit?usp=sharing) - let me know if your assigned partner is not here
2. Install Tinker from [thinkingmachines.ai](thinkingmachines.ai) following [these instructions](https://tinker-docs.thinkingmachines.ai/install).
   1. Install the Cookbook too.
   2. Sign into the [console](https://tinker-console.thinkingmachines.ai/) using your UVA email address.
   3. Set `TINKER_API_KEY` in your shell environment.
3. Read through the [Introduction to Fine-Tuning LLMs](finetuning_intro_lesson_plan.html)
4. Complete the [Fine-Tuning Exercise: Text-to-SQL with Tinker](sql_finetuning_lesson_plan.html)
